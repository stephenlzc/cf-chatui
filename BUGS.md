## 🔴 紧急 (Critical)

| ID | 问题 | 状态 | 影响 | 描述 | 临时解决 |
|:---:|------|:----:|------|------|----------|
| BUG-001 | GPT-OSS-120B 无响应 | 🔴 | 高 | 选择 GPT-OSS-120B 后发送消息无响应或卡住 | 使用 GLM-4.7-Flash 替代 |
| BUG-002 | Token 过期检查缺失 | 🔴 | 中 | JWT 过期后前端不会自动跳转登录页，显示 401 错误 | 手动刷新页面重新登录 |
| BUG-003 | SSE 流式中断 | 🟡 | 中 | 弱网环境下 SSE 连接可能意外中断 | 重新发送消息 |

**详细说明:**

### BUG-001: GPT-OSS-120B 模型响应问题
- **可能原因**: 该模型可能需要特定消息格式 / 流式响应不兼容 / 响应超时
- **建议修复**: 检查模型调用参数，添加特定处理逻辑

### BUG-002: Token 过期检查
- **可能原因**: 前端缺少 token 过期时间校验
- **建议修复**: 发送请求前检查 `exp` 字段，401 时自动清除 token

### BUG-003: SSE 流式响应中断
- **可能原因**: 网络不稳定导致连接断开
- **建议修复**: 添加 EventSource 重连机制和超时检测
